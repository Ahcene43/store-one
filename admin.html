إعدادات GitHub

كيفية الحصول على التوكن:

  1. اذهب إلى GitHub → Settings → Developer settings
  2. اختر Personal access tokens → Tokens (classic)
  3. انقر على Generate new token → Generate new token (classic)
  4. أعطِ اسم مثل "BEN&KRAB-Shopp"
  5. اختر "No expiration" للمدة
  6. اختر scope repo (كامل التحكم بالمستودعات)
  7. انسخ التوكن والصقه هنا
